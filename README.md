# MobileCPC - 手机端XCPC思维训练平台

一个简洁、高效、真正可用的算法训练平台。

## 🎯 项目特点
- ✅ **真正可用** - 所有功能都经过测试，可以直接使用
- ✅ **简洁设计** - 没有过度工程化，代码清晰易懂
- ✅ **易于部署** - 一键部署脚本，支持多种环境
- ✅ **完整文档** - 详细的开发和使用文档
- ✅ **开源友好** - MIT许可证，欢迎贡献

## 🚀 快速开始

### 环境要求
- Node.js 18+
- PostgreSQL 14+
- npm 8+

### 安装步骤
```bash
# 克隆项目
git clone https://github.com/suyunqiaoKID/MobileCPC.git
cd MobileCPC

# 安装依赖
./scripts/setup.sh

# 启动开发环境
./scripts/start-dev.sh
```

## 📁 项目结构
```
MobileCPC/
├── backend/          # 后端API服务
├── frontend/         # 前端Web界面
├── docs/            # 项目文档
├── scripts/         # 部署和工具脚本
└── tests/           # 测试文件
```

## 🔧 技术栈
- **后端**: Node.js + Express + TypeScript + Prisma
- **前端**: React + TypeScript + Vite + Tailwind CSS
- **数据库**: PostgreSQL
- **部署**: Docker + PM2 + Nginx
