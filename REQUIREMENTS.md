# MobileCPC - 真正可用的核心功能需求

## 🎯 项目定位
一个简洁、高效、真正可用的手机端XCPC思维训练平台

## ✅ 必须有的核心功能（MVP）

### 后端 (必须实现)
1. 用户认证系统（注册/登录）
2. 题目管理（增删改查）
3. 代码提交和评测
4. 学习进度跟踪
5. 基础数据统计

### 前端 (必须实现)
1. 用户登录/注册界面
2. 题目列表和详情
3. 代码编辑和提交界面
4. 个人进度展示
5. 响应式设计（支持手机）

### 数据库 (必须实现)
1. 用户表
2. 题目表
3. 提交记录表
4. 学习进度表

## ❌ 不需要的功能（暂时）
1. 复杂的推荐算法
2. 竞赛系统
3. 社交功能
4. 复杂的权限系统
5. 华而不实的UI效果

## 🚀 开发原则
1. 简洁优先
2. 功能可用性 > 美观
3. 代码可维护性
4. 易于部署
5. 良好的文档
